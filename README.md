# Money Transfer Tracker

This is a simple application to track money transfer transactions. It helps manage and organize all your transactions efficiently.

## Features
- Keep track of your income and expenses.
- Generate reports on your transactions.
- Easy to use and navigate.

## Installation
To install the application, clone the repository:
```bash
git clone https://github.com/mushiantipas/money-transfer-tracker.git
```

## Usage
- Open the application after installation.
- Follow the on-screen instructions to input your transactions.
